### Install Node.js

Download from ```https://nodejs.org/en/download/```

### Install npm

Download from ```https://www.npmjs.com/get-npm```

### Install React

In npm terminal:

```bash
npm install react
npm install react-dom
```

### Test React Application

In npm terminal:

```bash
# Create App using 
npx create-react-app ces_official_website
# Start testing using
npm start 
# Or
npm test
```

### Build React Application

In npm terminal:

```bash
npm build
```
